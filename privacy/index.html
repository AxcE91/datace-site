Privacy Policy

Ultimo aggiornamento: maggio 2026  ·  Versione 2.0  ·  Valida per tutti i servizi Datace

I tuoi documenti non vengono salvati sui server Datace.

La maggior parte degli strumenti (F24 in Excel, Fattura elettronica in Excel, Archivio Fatture AdE, Scadenze F24) elabora i file direttamente nel browser, senza inviarli ai nostri server. Fa eccezione la Busta paga spiegata: il cedolino viene inviato all'API di Google Gemini per generare la spiegazione richiesta. Non utilizziamo database per conservare i documenti caricati.

2. Ambito di applicazione

La presente informativa si applica a tutti i servizi Datace: F24 in Excel, Fattura in Excel, Archivio Fatture AdE, Scadenze F24, Busta paga spiegata e qualsiasi futuro strumento del brand.

3. Trattamento dei file caricati

Datace è progettato per ridurre al minimo la conservazione dei documenti caricati. In concreto:

  • ✓gli strumenti client-side (F24 in Excel, Fattura elettronica in Excel, Archivio Fatture AdE, Scadenze F24) elaborano i file direttamente nel browser: i documenti non vengono caricati né salvati sui server Datace;
  • ✓la Busta paga spiegata invia il PDF del cedolino all'API di Google Gemini, necessaria per generare la spiegazione richiesta;
  • ✓non utilizziamo database per conservare i documenti caricati;
  • ✓i dati tecnici minimi necessari al funzionamento sono gestiti dall'infrastruttura di hosting.

4. Dati trattati

File caricati — negli strumenti client-side sono elaborati nel browser, senza upload verso i server Datace. Per la Busta paga spiegata il PDF viene inviato all'API di Google Gemini per produrre la spiegazione. Base giuridica: esecuzione del servizio (art. 6.1.b GDPR). I documenti non vengono usati da Datace per addestrare modelli AI.

Dati tecnici (log) — IP, user agent, data/ora. Gestiti dall'infrastruttura di hosting. Base giuridica: legittimo interesse (art. 6.1.f GDPR).

Contatti volontari — email e messaggi inviati spontaneamente. Conservati fino a risoluzione della richiesta, non oltre 24 mesi. Base giuridica: consenso (art. 6.1.a GDPR).

5. Cookie e tracciamento

I servizi Datace non utilizzano cookie di profilazione o tracciamento pubblicitario: nessun Google Analytics, Meta Pixel o strumenti simili. Per le statistiche di utilizzo usiamo Umami Cloud, una soluzione di analytics privacy-friendly e senza cookie, che raccoglie dati aggregati e non costruisce profili individuali. Possono essere presenti cookie tecnici strettamente necessari al funzionamento della sessione.

6. Servizi terzi e infrastruttura

Non vendiamo né cediamo i tuoi dati per finalità di marketing. Alcuni fornitori tecnici ricevono i dati necessari a erogare il servizio:

  • ·Railway — infrastruttura e hosting che mantengono attivi i webtool, USA. Trasferimento extra-UE su base SCC.
  • ·GitHub Pages — hosting del sito statico datacetools.com, USA. Trasferimento su base SCC.
  • ·Google LLC — Google Fonts: l'indirizzo IP può essere trasmesso per scaricare il font.
  • ·Tailwind CDN — caricamento della libreria CSS per l'interfaccia: l'indirizzo IP può essere trasmesso per scaricare la risorsa.
  • ·Umami Cloud — statistiche di utilizzo aggregate, analytics privacy-friendly e senza cookie.
  • ·Google Gemini API — usata solo dallo strumento Busta paga spiegata. Secondo i termini del servizio a pagamento, i dati inviati tramite API non vengono usati per addestrare o migliorare i prodotti Google. Trasferimento su base SCC.
