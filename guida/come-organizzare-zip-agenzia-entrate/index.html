Come organizzare il ZIP di Fatture e Corrispettivi

Guida passo passo per elaborare il file ZIP scaricato dal portale dell'Agenzia delle Entrate e ottenere un Excel strutturato con le fatture organizzate per anno e mese.

Cos'è il file ZIP di Fatture e Corrispettivi

Il portale Fatture e Corrispettivi dell'Agenzia delle Entrate permette di scaricare in blocco tutte le fatture elettroniche ricevute e inviate tramite il Sistema di Interscambio (SdI). Il download avviene in formato ZIP: un archivio che contiene decine o centinaia di file XML (le fatture), file .xml.p7m (fatture firmate digitalmente) e notifiche SdI.

Il problema è che quel ZIP arriva in forma grezza: i file hanno nomi tecnici assegnati dallo SdI (es. IT01234567890_FPR01.xml), senza alcuna organizzazione per data, fornitore o tipo. Per usare quei dati in modo pratico serve estrarli, classificarli e portarli in Excel.

Cosa serve prima di iniziare

  • Il file ZIP scaricato da Fatture e Corrispettivi (max 100 MB)
  • Opzionalmente: la tua P.IVA o Codice Fiscale per distinguere fatture attive e passive
  • Un browser aggiornato (Chrome, Firefox, Edge, Safari)

Passo 1 — Scarica il ZIP da Fatture e Corrispettivi

Accedi al portale Fatture e Corrispettivi con le credenziali Fisconline o SPID. Nella sezione dedicata seleziona il periodo e scarica il file ZIP sul tuo computer. Il file può contenere sia fatture attive (emesse) che passive (ricevute), notifiche SdI e metadati.

Passo 2 — Carica il file su Archivio Fatture AdE

Apri Archivio Fatture AdE. Trascina il file ZIP nell'area di caricamento oppure clicca per selezionarlo.

Se vuoi che il sistema distingua automaticamente le fatture attive da quelle passive, inserisci la tua P.IVA o Codice Fiscale nel campo apposito. Il sistema confronterà cedente e cessionario di ogni fattura con il tuo identificativo per classificarla come attiva o passiva quando i dati sono sufficienti.

Passo 3 — Elaborazione automatica

Archivio Fatture AdE apre il ZIP in memoria e, per ogni file XML al suo interno:

  • Gestisce fatture per privati, pubblica amministrazione e file con firma digitale
  • Rileva il tipo di documento (fattura, nota di credito, parcella, autofattura...)
  • Estrae fornitore, cliente, data, importo totale
  • Classifica la fattura come Attiva, Passiva o Unknown
  • Riconosce e organizza le notifiche SdI e le separa dalle fatture

Passo 4 — Scarica il risultato

Ricevi un unico file ZIP con due contenuti:

  • fatture_organizzate.xlsx: una riga per fattura con denominazione, P.IVA/CF, data, importo, tipo (Attiva/Passiva) e nome file originale
  • archivio_organizzato.zip: tutti i file XML riorganizzati in cartelle ANNO/Mese/nomefile.xml, con le notifiche in una cartella /Notifiche/ separata

Problemi comuni e come risolverli

ZIP protetto da password

Alcuni ZIP scaricati da portali gestionali o intermediari sono protetti da password. Archivio Fatture AdE non può elaborarli: rimuovi la protezione con un software come 7-Zip o WinRAR prima di caricare il file.

Fatture classificate come "Unknown"

Accade quando la struttura delle cartelle nel ZIP non contiene parole chiave riconoscibili (come "emesse", "ricevute") e non è stata inserita la P.IVA. In questo caso le fatture vengono comunque estratte e inserite nell'Excel: manca solo la classificazione attiva/passiva. Inserire la P.IVA risolve il problema.

Il ZIP non contiene fatture riconoscibili

Se il ZIP non contiene file XML nel formato FatturaPA, Archivio Fatture AdE restituisce un errore. Verifica che il file sia il download diretto da Fatture e Corrispettivi e non un archivio generico.

Archivio Fatture AdE supporta

  • ✓ ZIP da Fatture e Corrispettivi AdE
  • ✓ FPR12, FPA12, .xml.p7m
  • ✓ Notifiche SdI (RC, NS, MC, EC, DT)
  • ✓ Classificazione attive/passive
  • ✓ Archivio organizzato Anno/Mese
  • ✓ ZIP fino a 100 MB

Domande frequenti

Come si organizza il ZIP di Fatture e Corrispettivi?

Apri Archivio Fatture AdE e carica il file ZIP. Archivio Fatture AdE analizza automaticamente tutti gli XML interni e restituisce un Excel strutturato e un archivio organizzato per anno e mese.

Come distinguo fatture attive e passive nel ZIP AdE?

Archivio Fatture AdE rileva il tipo dalla struttura delle cartelle nel ZIP. Se le cartelle non sono chiare, inserisci la tua P.IVA/CF nell'apposito campo: il sistema confronta cedente e cessionario di ogni fattura per classificarla come attiva o passiva quando i dati sono sufficienti.

I miei file vengono salvati?

File elaborati temporaneamente in memoria. Datace non conserva i documenti caricati. L’elaborazione serve solo a generare l’output.

Sono supportate le notifiche SdI nel ZIP?

Sì. Archivio Fatture AdE riconosce e gestisce le notifiche SdI (RC, NS, MC, EC, DT) presenti nel ZIP, inserendole in una cartella /Notifiche/ dedicata nell'archivio organizzato.
