La Fattura Elettronica XML: struttura, tracciato e come gestire i dati in Excel

Guida pratica per commercialisti e responsabili amministrativi

Perche convertire le fatture XML in Excel

Il file XML della fattura elettronica e pensato per essere letto dai software, non dalle persone. Per studi e uffici amministrativi che gestiscono centinaia di fatture, averle in formato Excel apre possibilita operative concrete:

  • ·Controllo prima della liquidazione IVA: verificare imponibili e imposte per aliquota su tutte le fatture del periodo.
  • ·Riconciliazione contabile: i dati strutturati sono pronti per essere incrociati con le registrazioni contabili tramite CERCA.VERT o Power Query.
  • ·Analisi fornitori e clienti: raggruppare il fatturato per controparte e individuare i principali rapporti commerciali.
  • ·Controllo scadenze: ordinare i pagamenti per data di scadenza e gestire l'incassato e lo scaduto.

La struttura del tracciato FatturaPA

Il file XML segue lo schema FatturaPA ed e diviso in due grandi blocchi: l'Header con le anagrafiche e il Body con i dati del documento.

Cedente / Prestatore

Il fornitore che emette la fattura: denominazione, partita IVA e codice fiscale.

Cessionario / Committente

Il cliente destinatario: denominazione, partita IVA o codice fiscale.

Dettaglio Linee

Le singole righe: descrizione, quantita, prezzo unitario e totale, aliquota IVA.

Dati Riepilogo

Imponibile e imposta per ogni aliquota, con natura ed esigibilita IVA.

Dati Pagamento

Modalita di pagamento (bonifico, RIBA, SEPA, contanti...) e date di scadenza degli importi.

XML semplice o file .p7m firmato

Le fatture elettroniche circolano in due formati: il semplice .xml e il .xml.p7m, ovvero l'XML racchiuso in una busta di firma digitale. Entrambi contengono gli stessi dati: FatturaXLS li elabora entrambi automaticamente, senza bisogno di software aggiuntivi.

Domande frequenti

Come si converte una fattura elettronica XML in Excel?
Carica il file XML su fattura.datacetools.com, il sistema legge il tracciato FatturaPA ed estrae automaticamente tutti i dati, scaricando un file Excel strutturato su 5 fogli. Bastano pochi secondi, senza installazioni.
Sono supportati i file .p7m firmati digitalmente?
Si. Sono supportate sia le fatture XML semplici sia quelle .xml.p7m firmate digitalmente, come quelle scaricate dallo SdI o dal cassetto fiscale.
I file delle fatture vengono salvati?
No. I file vengono elaborati esclusivamente in memoria RAM e cancellati immediatamente dopo la risposta. Nessun dato viene scritto su disco o trasmesso a terzi.
Posso convertire piu fatture insieme?
Si, fino a 50 file contemporaneamente. Tutte le fatture vengono unificate in un unico Excel. Sono gestiti anche i file XML che contengono piu fatture (lotti): ogni fattura finisce su una riga separata.
Quali dati vengono estratti dalla fattura?
Fornitore e cliente con partita IVA e codice fiscale, tipo, numero e data del documento, le righe di dettaglio con quantita e prezzi, i riepiloghi IVA per aliquota con natura ed esigibilita, e le scadenze di pagamento.
In quanto tempo viene generato l'Excel?
In meno di un minuto. Il file viene elaborato in memoria RAM e il download parte automaticamente non appena la conversione e completata.
