Come funziona

1

Carica i file XML

Ricevi un file .xlsx con 3 fogli: Riepilogo Fatture, Dettaglio Righe, Riepilogo IVA.

Cosa estrae FatturaXLS

Anagrafiche

  • Fornitore (cedente/prestatore) con P.IVA e CF
  • Cliente (cessionario/committente) con P.IVA e CF

Documento

  • Tipo (TD01 fattura, TD04 nota di credito, ecc.)
  • Numero, data, importo totale

Righe di dettaglio

  • Descrizione, quantita, prezzo unitario e totale
  • Aliquota IVA e natura per ogni riga

IVA e pagamenti

  • Riepilogo IVA per aliquota con natura ed esigibilita
  • Modalita e scadenze di pagamento

Domande frequenti

Come si converte una Fattura Elettronica XML in Excel?

Vai su fattura.datacetools.com, carica il file XML o .p7m, attendi l'estrazione automatica e scarica il file Excel con tutti i dati strutturati su 3 fogli.

Sono supportati i file .p7m firmati?

Si. FatturaXLS supporta sia i file XML semplici sia i file .xml.p7m firmati digitalmente (CAdES), come quelli scaricati dallo SdI o dal cassetto fiscale.

I file vengono salvati?

No. Ogni file viene elaborato interamente in memoria RAM. Nessun dato viene scritto su disco o trasmesso a terzi. I dati vengono eliminati non appena l'Excel viene inviato al browser.

Posso caricare piu fatture insieme?

Si, fino a 50 file XML o .p7m contemporaneamente. Tutte le fatture vengono consolidate in un unico Excel con un foglio di riepilogo cumulativo.
